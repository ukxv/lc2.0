# Imkon LC — Учебный Центр

## Запуск

```bash
# 1. Установить зависимости
npm install --ignore-scripts

# 2. Запустить сервер
node server.js

# 3. Открыть в браузере
http://localhost:3000
```

## Функционал

### Администратор
- 📊 Дашборд: доходы центра, учителей, балансы студентов
- 👥 Управление пользователями (учителя, студенты)
- 📚 Группы: создание, назначение учителей и студентов
- 📅 Расписание: дни, время, кабинет
- ✅ Посещаемость учителей и студентов
- 💳 Пополнение баланса (40% → учителю, 60% → центру)
- 💰 Настройка ставки каждому учителю отдельно
- 🛒 Магазин: добавление товаров с фото и ценой в коинах
- 📦 Управление заказами (Ожидает / Подтверждён / Доставлен / Отменён)
- 🔄 Сброс баланса в начале месяца

### Учитель
- 📊 Свой доход, студенты, уроки за неделю
- 📅 Расписание на неделю
- 📚 Группы: создание уроков (тема + ДЗ), отметка посещаемости
- 🪙 Выдача коинов студентам

### Студент  
- 📊 Баланс и коины
- 📅 Расписание с темами и домашними заданиями
- 🛒 Магазин: покупка товаров за коины
- 📦 История заказов
